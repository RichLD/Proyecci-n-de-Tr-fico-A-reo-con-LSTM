Este proyecto implementa un modelo de Deep Learning utilizando redes neuronales recurrentes de tipo LSTM (Long Short-Term Memory) para predecir el flujo mensual de pasajeros en tráfico aéreo. El sistema procesa datos históricos, realiza ingeniería de variables avanzadas (incluyendo codificación cíclica) y genera proyecciones con alta precisión para optimizar la toma de decisiones operativas y logísticas.
